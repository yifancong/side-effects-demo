
# rspack-minimal-standalone

This project is a minimal standalone example using Rspack and React, synced to https://github.com/yifancong/side-effects-demo.

## Project Structure

- `src/`: Main source code, including components, pages, and styles
- `public/`: Static assets
- `index.html`: Entry HTML file
- `rspack.config.js`: Rspack configuration
- `tsconfig.json`: TypeScript configuration
- `.gitignore`: Git ignore file

## Usage

1. Install dependencies:
   ```sh
   pnpm install
   ```
2. Build the project:
   ```sh
   pnpm run build
   ```
3. Start the development server:
   ```sh
   pnpm run dev
   ```
